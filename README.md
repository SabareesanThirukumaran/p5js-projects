# p5js-projects
### Computational Art & Mathematical Simulations

A curated collection of generative sketches exploring coordinate geometry, noise algorithms, and 3D projection using the p5.js library.

[Live Gallery](https://sabareesanthirukumaran.github.io/p5js-projects/)

---

## 🔬 Core Experiments

### 01. Torus Simulation
* **Concepts:** Trigonometric mapping, 3D Rotation Matrices.
* **Logic:** Utilizes nested loops to map a 2D plane onto a 3D toroidal surface ($x, y, z$).

### 02. Star Wars Intro
* **Concepts:** Particle Systems, Z-depth scaling.
* **Logic:** Implements a perspective projection where $z$-coordinates are decremented to simulate forward velocity.

### 03. Perlin Noise Simulator
* **Concepts:** Flow Fields, Scalar Noise.
* **Logic:** Employs Perlin noise to generate a vector field, dictating the movement of autonomous agents.
